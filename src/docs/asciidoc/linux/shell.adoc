== 常用软件安装脚本

本提供了五个脚本,具体说明如下

* deploy.sh : 主脚本
* docker.sh : docker安装脚本
* native.sh : 本地安装脚本
* unnative.sh : 本地卸载脚本
* undocker.sh : docker 卸载脚本

脚本中涉及到的目录

[[linux-shell-install-tbl]]
.默认目录
|===
| 变量名 | 目录 | 说明

| BASH_URL | http://software.jcohy.com/linux/ | 所有所需软件安装包所在的服务器(可以使用 httpd 或者 nginx 等服务器搭建)

| BASH_CONFIG | http://software.jcohy.com/bash/ | 安装所需脚本和配置文件目录

| BASE_DIR | /opt/software | 本地存放安装包的地址

| DOCKER_DIR | /usr/local/docker | docker 安装软件的目录

| INSTALL_DIR | /usr/local | 本地安装地址
|===

软件版本可查看 <<linux-install-tbl>>
